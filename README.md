## 🛠️ Featured Project: GatorMart (Campus Marketplace Ecosystem)

As the <b>Backend Lead</b>, I translated software engineering rigor into system stabilization, taking ownership of the full-stack data flow and production-level infrastructure setup:

* 🏗️ **Backend Architecture & Full-Stack Security**: Designed and initialized the core **FastAPI** structure. Implemented an end-to-end secure **Auth Workflow** featuring domain-level restrictions (`@sfsu.edu`), secure password hashing via **bcrypt**, **JWT Token** handling, and granular protected-route interception logic.
* 🗄️ **Relational Database Design**: Designed and evolved robust MySQL database schemas spanning relational mappings for users, posts, instant messaging (`conversations`/`messages`), and status state machines (`post_status`, `request_status`). Enforced **parameterized queries** at the code level to completely eliminate SQL injection vulnerabilities.
* ☁️ **AWS Production DevOps & Server Configuration**: Led the deployment phase on **AWS EC2 Linux** instances. Configured **NGINX** as a reverse proxy to eliminate cross-origin resource sharing (CORS) overhead, guaranteeing seamless, low-latency communication between the React frontend and FastAPI backend under a unified domain.
* 🎛️ **Multi-Module Integration & Stabilization**: Functioned as the core engineering pillar during deadline sprints, resolving multi-module integration blockers including routing inconsistencies, frontend/backend asynchronous state mismatches, and API connection failures. Engineered a robust media upload infrastructure utilizing **multipart data streams, strict format validation, and UUID-based file renaming** for secure storage.
